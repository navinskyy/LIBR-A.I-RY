# 📚 LIBR-A.I-RY

**LIBR-A.I-RY** is a modern web-based library management system designed to simplify the management of library resources. It provides an intuitive interface for both librarians and users, making book cataloging, borrowing, returning, and inventory management more efficient.

## ✨ Features

- 📖 Browse and search the book catalog
- 🔍 Advanced search functionality
- 📚 Book borrowing and return management
- 👤 User authentication and account management
- 📝 Book reservation system
- 📊 Library statistics and reports
- 📱 Responsive design for desktop and mobile devices
- ⚡ Fast and user-friendly interface

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL

## 🚀 Getting Started

### Prerequisites

- XAMPP or any PHP development environment
- PHP 8.0+
- MySQL
- Modern web browser

### Installation

1. Clone the repository

```bash
git clone https://github.com/navinskyy/LIBR-A.I-RY.git
```

2. Move the project into your web server directory (e.g., `htdocs` for XAMPP).

3. Import the provided database into MySQL using phpMyAdmin.

4. Update the database configuration if necessary.

5. Start Apache and MySQL.

6. Open your browser and navigate to:

```
http://localhost/LIBR-A.I-RY
```
